# Facility A inspection runbook

## Purpose

Short inspection checklist for forecast-driven incidents.

## Inspection order

1. Confirm the operating mode proxy.
2. Check the summed 15-step forecast and the residual threshold.
3. Review top feature shifts around the top episode.
4. Inspect pressure, temperature, compressor, and flow signals in that order.

## When the forecast is below expected

- In gas-like mode, first verify that zero heat is not expected behavior.
- In heating-like mode, inspect inlet temperature, return temperature, and compressor load.

## When the forecast cost is rising

- Compare the predicted 15-step sum against the assumed gas and electric rates.
- Escalate if the predicted operating cost moves outside the expected band.

## Notes

- This runbook is synthetic and meant for the project RAG corpus.
